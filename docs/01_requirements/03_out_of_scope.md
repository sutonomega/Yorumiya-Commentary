# Out of Scope

MVP では次を必須にしない。

## UI / Product Surface

- GUI application
- OBS overlay
- browser extension
- mobile application

これらは pipeline が安定してから integration として扱う。

## Production Model Quality

- 高精度な scene understanding
- 完全な UI recognition
- 多人数会話の正確な speaker separation
- 人間感情の詳細分類

MVP では adapter boundary と fallback behavior を優先する。

## Advanced Companion Behavior

- long-term semantic memory
- personality growth
- complex dialogue planning
- full-duplex voice conversation

Companion behavior は M5 の主題とし、MVP では skeleton に留める。

## Persistent Infrastructure

- cloud database
- distributed queue
- authentication
- account system
- multi-device sync

初期段階では local runtime と in-memory state を基本にする。
